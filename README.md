## IronTree Axum

> axum 后台快速开发框架后端

### 技术选型

- Axum：Tokio 开源 web 框架

- MongoDB：非关系行数据库

- moka-rs：快速、并发的 Rust 缓存库

## Tip

### 菜单权限模型

```json
{
  "父级菜单": "父级菜单id",
  "菜单名称": "name",
  "创建时间": "time",
  "排序ID": "id",
  // 菜单;按钮
  "菜单类型": "type",
  "菜单图标": "选择图标",
  // 菜单:前端页面地址;按钮:后端api地址
  "router路由地址": "txt",
  "权限标识": "api权限标识"
}
```
